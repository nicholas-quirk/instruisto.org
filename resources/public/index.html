For Language Learners

I first came across Esperanto when I was in college. I wanted to learn a language for fun, and I think like many in my position Google searched for "easiest language to learn" or "easiest language for native English speakers to learn". Somewhere in those search results Esperanto was mentioned and I checked it out. I tried learning it from lernu.net. I eventually gave up, probably due to my college work load and a frustration with myself for not learning faster enough. Fast forward three to four years later (2012). I'm a full time programmer learning new programming technologies in my spare time. I'm constantly reading about new programming languages and working on side projects. I'm starting to feel brittle. My thinking is all one sided, it's all logic. I needed something different to flex my brain. I started the same Google searches again, and landed on the same language. Esperanto was the only one with a philosophy. It was a language that had a purpose.

I think anyone coming to this language would share the same sentiment. Anyone interested in learning Esperanto would want to learn a language to support the greater good. To learn something that was created for the whole world. It's an international auxiliary language that facilitates openness.

What this site is: This site is meant to be an introduction I wish I had. I wanted to create a concise, easy-to-use, step-by-step approach. All one has to do is follow each module down the list. The reading modules should take no more than 5 minutes and the review modules should take no more than 10-15 minutes. I also want this site to be a review/reference guide as you further your learning. Please check out the additional resources on the site. I've included the ones that I think I would benefit you the most in the least amount of time.

What this site is not: This site is not a complete course of Esperanto. It's not an experts approach at teaching people the language. At its current state, I make no claim that this will make you proficient in reading, writing or speaking Esperanto. Think of it as a place to wet your appetite and review concepts.

For Programmers

From the users point of view, this is a basic HTML/JavaScript site. In actuality, this site is written in ClojureScript. I came to Clojure a year ago while exploring functional programming languages. Being a Java developer, I was very interested in this new JVM language. I wrote my first personal website in Clojure, and found ClojureScript to be intriguing.

This site has two purposes. The first being an introduction site to Esperanto. One in which I think the community very much needs, even though this may not be that site. Secondly, this is a proof of concept at least to myself that one can write a single page application. This is a very basic proof. Most of the material is contained within the HTML, but the state changes and dynamic features are written with ClojureScript which compiles to JavaScript.

This has been a labor of intellectual pursuits. I've been selfish in building this site around a new technology that may not last, or that I may lose interest in. I hope this will at least spark interest in language learners wanting to learn Esperanto, or programmers that should give functional programming a chance (even though my ClojureScript isn't very functional).

As a side note, this site by it's nature solves two problems which I find interesting. The first one is that it can be used entirely off-line. Once all the pages are cached (which aren't many), the site is still completely functional. Secondly, because you get the site in one download from the start there is no drain on mobile devices batteries. All network usage required for the site is paid on the initial page load.
